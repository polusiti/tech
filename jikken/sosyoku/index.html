function quickSort(arr) {
    if (arr.length <= 1) return arr;
    const pivot = arr[Math.floor(arr.length / 2)];
    const left = arr.filter(x => x < pivot);
    const middle = arr.filter(x => x === pivot);
    const right = arr.filter(x => x > pivot);
    return [...quickSort(left), ...middle, ...quickSort(right)];
}

tcolorbox 使用ガイド

<div class="tcolorbox"
     data-tcolorbox-title="タイトル"
     data-tcolorbox-subtitle="サブタイトル">
    ここにコンテンツを記述
</div>

利用可能な色のバリエーション：

  • 基本: class="tcolorbox" (灰色)
  • 青色: class="tcolorbox tcolorbox-blue"
  • 赤色: class="tcolorbox tcolorbox-red"
  • 定理・定義: 数学・科学の定理を強調
  • コード例: プログラミングのヒントや例示
  • 重要注意: 赤色を使って警告や注意を表示
  • 補足情報: 青色で追加情報を提供
  • 引用: 文献や名言の引用

現在の実装での注意点：

  • タイトルが長すぎる場合は自動的に省略されます
  • ネストされたtcolorboxは非推奨です
  • モバイル表示では装飾が少し簡略化されます
